## 项目打包

1.修改工程根目录的`gradle.properties`中的`isNeedPackage=true`。

2.添加并配置keystore，在`versions.gradle`中修改`app_release`相关参数。

3.使用友盟统计，在`local.properties`中设置应用的友盟ID:`APP_ID_UMENG`。
